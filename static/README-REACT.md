# React + TailwindCSS 前端升级说明

## 概述

这是一个使用 React + TailwindCSS + Framer Motion 重构的前端版本，保留所有原有功能，并添加了主题切换和丰富的动画效果。

## 文件说明

- `index-react.html` - 新的 HTML 入口文件（使用 CDN 方式引入 React、TailwindCSS）
- `app-react.jsx` - React 应用主文件（包含所有组件和逻辑）
